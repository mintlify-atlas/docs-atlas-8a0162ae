Thoughts are Echo's memory system. Every link, note, or piece of context you save becomes part of Echo's knowledge base, enriching the actions it can perform on your behalf.

## What Are Thoughts?

Thoughts are pieces of information you want Echo to remember:
- **Links**: Articles, documentation, references
- **Notes**: Ideas, reminders, context
- **Screenshots**: Visual information (via share extension)
- **Context**: Anything relevant to your work or projects

<Info>
Echo automatically fetches metadata (title, description, preview) for shared links to make them more useful.
</Info>

## Why Save Thoughts?

Thoughts enhance Echo's proactive capabilities:

### Better Actions
When proposing actions, Echo considers your saved thoughts for:
- Relevant context and background
- Referenced documentation
- Project-specific information
- Your preferences and priorities

### Centralized Memory
Thoughts serve as a unified knowledge base across all your tools:
- Find information without remembering which tool you used
- Search across all saved content
- Organize with collections

### Share from Anywhere
Capture information as you discover it:
- Share from Safari, Chrome, or any browser
- Save from social media apps
- Capture from note-taking apps

## Viewing Thoughts

The Thoughts tab displays all your saved items:

### Search Bar
Quickly find specific thoughts:
- Type keywords to filter results
- Searches titles, descriptions, and notes
- Results update as you type
- Clear button appears when text is entered

### Collections
Thoughts can be organized into collections for easy access:
- Stacked cards show your collections at the top
- Tap a collection to view its thoughts
- Collections are based on domains or manual organization

### Thought Cards

Each thought displays:
- **Icon**: Represents the content type (link, document, etc.)
- **Title**: Page title or your custom label
- **Description**: Automatically fetched from the link's metadata
- **Note**: Your personal annotation (if added)
- **Read Status**: Visual indicator showing read/unread state
- **Archive Status**: Badge if the item is archived

<Tip>
Thought cards show more details when multiple items are from the same domain, helping you distinguish between similar links.
</Tip>

## Adding Thoughts

There are multiple ways to save thoughts to Echo:

### In-App Addition

<Steps>
  <Step title="Tap the Plus Button">
    In the Thoughts tab, tap the "+" button in the top-right corner.
  </Step>

  <Step title="Enter URL and Note">
    - Paste or type a URL
    - Add an optional note for context
    - Echo fetches metadata automatically
  </Step>

  <Step title="Save">
    Tap "Add Item" to save the thought.
  </Step>
</Steps>

### Share Extension (iOS)

<Steps>
  <Step title="Share from Any App">
    When viewing content you want to save:
    - Tap the Share button
    - Select "Echo" from the share sheet
  </Step>

  <Step title="Add Context (Optional)">
    The Echo share extension appears:
    - URL is pre-filled
    - Add a note if desired
    - Tap "Share" to save
  </Step>

  <Step title="Confirmation">
    Echo confirms the thought was saved and you return to your previous app.
  </Step>
</Steps>

### Share Intent (Android)

Android users can share to Echo from any app:
1. Tap the Share button in any app
2. Select "Echo" from the share menu
3. Add optional notes
4. Confirm to save

<Note>
The share extension captures the URL and any selected text automatically.
</Note>

## Viewing Thought Details

Tap any thought card to open the detail modal:

### Detail View Elements

**Header**
- Full title
- Source domain
- Save timestamp

**Preview**
- Open Graph image (if available)
- Rich preview of the content

**Description**
- Automatically fetched metadata
- Provides context about the link

**Your Note**
- Personal annotation you added
- Editable after saving

**Actions**
- Mark as read/unread
- Archive the thought
- Delete permanently
- Open in browser

## Managing Thoughts

### Swipe Actions

Swipe left or right on any thought card for quick actions:

**Swipe Right**
- Mark as read (if unread)
- Mark as unread (if read)
- Toggle read status with one gesture

**Swipe Left**
- Delete the thought
- Confirmation required before deletion

<Warning>
Deleted thoughts cannot be recovered. Archive instead if you might need the thought later.
</Warning>

### Mark as Read/Unread

Track which thoughts you've reviewed:
- **Unread**: New or pending review
- **Read**: Already reviewed
- Visual opacity change indicates read status

### Archive Thoughts

Remove thoughts from active view without deleting:
1. Open thought detail modal
2. Tap "Archive" action
3. Thought moves to archive
4. Access archived thoughts from Archive screen

<Tip>
Archive thoughts that are no longer actively relevant but might be useful for reference later.
</Tip>

### Delete Thoughts

Permanently remove a thought:
1. Swipe left on the thought card, or
2. Open detail modal and tap "Delete"
3. Confirm deletion in the alert

## Collections

Organize related thoughts into collections:

### Automatic Collections
Echo automatically groups thoughts:
- **Domain-based**: Links from the same website
- **Smart grouping**: Related content detection

### Manual Collections
Create custom collections:
1. Navigate to Collections screen
2. Tap "New Collection"
3. Name your collection
4. Add thoughts by domain or manually

### Viewing Collections

<Steps>
  <Step title="Access Collections">
    - Tap a collection card in the Thoughts tab, or
    - Navigate to the Collections screen
  </Step>

  <Step title="Browse Collection Items">
    See all thoughts belonging to that collection.
  </Step>

  <Step title="Manage Collection">
    Add or remove thoughts from the collection.
  </Step>
</Steps>

## Search and Filter

### Searching Thoughts

The search bar enables quick filtering:
- Type keywords from title or description
- Search your personal notes
- Results update in real-time
- Search works across all active thoughts

### Clear Search
- Tap the "X" button in the search bar
- Returns to full thoughts list
- Collections reappear

<Note>
Search only includes active thoughts, not archived ones.
</Note>

## Pull to Refresh

Update your thoughts list:
1. Pull down on the thoughts list
2. Release to refresh
3. Latest thoughts and metadata updates appear

## Best Practices

### Add Context with Notes
- Always add notes to links for future reference
- Explain why you saved the thought
- Include project names or relevant tags
- Add your own summary or key takeaways

### Regular Review
- Periodically review unread thoughts
- Archive or delete thoughts that are no longer relevant
- Update notes as context changes

### Use Collections
- Organize thoughts by project or topic
- Create collections for ongoing work
- Review collection contents regularly

### Share Liberally
- Save anything that might be useful to Echo
- Over-sharing is better than under-sharing
- Echo intelligently uses relevant context

## Integration with Actions

Thoughts directly enhance Echo's actions:

### Context for Proposals
When Echo proposes an action, it considers:
- Related thoughts from the same domain
- Notes mentioning relevant keywords
- Recently saved thoughts about the project

### Referenced in Timeline
Action timelines may reference specific thoughts:
- See which thoughts informed the action
- Understand Echo's reasoning
- Verify context was used correctly

## Troubleshooting

### Metadata Not Loading
- Some websites block metadata fetching
- Add a manual note to provide context
- The thought is still saved with the URL

### Share Extension Not Working
- Ensure Echo app is installed and signed in
- Restart the sharing app
- Reinstall Echo if issues persist

### Thoughts Not Syncing
- Check internet connection
- Pull to refresh in the Thoughts tab
- Sign out and back in if sync fails

### Search Not Finding Items
- Verify spelling of search terms
- Check if thought is archived
- Search matches title, description, and notes only
